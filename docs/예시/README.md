## 파일 목록

### 1. 견적서 (1개)

- **`태양광 예상견적서(봉남리6-2) A3.pdf`**
- 양식: A3 가로
- 내용:
  - 좌측 = 20년 cash-flow 표 (년차/SMP/REC/총수익/유지보수/순수익/월평균/누적/대출상환/잔여)
  - 우측 = 입력 변수 박스
    - 용량 (kW), 일발전시간 (h), SMP 단가, REC 가중치, REC 단가
    - 시공비 (총액), 자기부담, 대출액, 금리(%), 거치기간(개월), 상환기간(년)
    - ROI(%), 회수기간(년), 20년 누적 순익
- Phase 4 견적서 PDF 의 정확한 청사진

### 2. 배치도 (2개 — 양식 참고용)

- **`배치도(전라남도 구례군 구례읍 봉남리 6-2).pdf`** (267kW 사례)
- **`배치도(전라남도 순천시 용당동 118-1).pdf`** (58kW 사례)
- 양식 구조 (양쪽 동일):
  - 좌측 큰 영역 = 카카오맵/항공뷰 캡처 + 패널 그리드 시각화 + 나침반(N/S/E/W)
  - 우측 상단 = 회사 로고 (Solar LD inc)
  - 우측 본문:
    - **부지** (지번)
    - **모듈** (예: AIKO 670Wp 2,465 × 1,134 × 33mm)
    - **패널 수량** (개)
    - **총 용량** (kW = 패널W × 수량 ÷ 1000)
    - **이격거리** (m)
    - **인버터** (사양 미상 — 의뢰자 추가 확인 필요)
  - 우측 하단 = 회사 정보

    - 팩스(?): 0504) 370-2940
    - 이메일: slddettkh@naver.com

### 3. Excel 견적서 (1개 — 보류)

- **`서면 압곡리 1002-12-1.xlsx`**
- 시트: `일출1.5` (REC 가중치 1.5 시나리오) / `견적서` (요약) / `일출1` (가중치 1.0 시나리오)
