# 의뢰자 사용 피드백 (2026-05-02 카톡)

> 썬맵 이것저것 사용하고 있는데 아쉬운점이
> 1. 지도상에 필지 누르고 있으면 빨간색음영 정보 나오면 좋겠어요
>    공매 들어가서 필지 정보확인하는거 말구 바로클릭하면나오게요
> 2. 지도에 전기+ 공매 활성화되면 전기선로표시기+공매(빨간색)표시 같이나오면 좋겠어요
> 3. 지도에서 전기+ 경매 활성화되면 전기선로표기+경매(노란색 표시가 같이나오면좋겠어요
>
> 나머지 부분은 아주 맘에듭니다

**전제**: "나머지는 맘에 든다" = 시연 통과의 신호. 이 3가지 마무리되면 의뢰자 만족도 상승.

## 1. 필지 클릭 시 빨간 음영 (즉시 시각 피드백)

### 의뢰자 의도
- 현재 흐름: 공매 모드 진입 → 매물 마커 클릭 → 우측 패널에서 필지 정보 확인
- 원하는 흐름: **지도 어디서든 필지 클릭 → 즉시 그 필지에 빨간 음영 표시 → 우측 패널 자동 열림**
- "공매 들어가서 ~ 말구" = 공매 모드 OFF 상태에서도 필지 클릭만으로 정보 확인 가능해야 함

### 현재 상태 추정 (코드 확인 필요)
- 메인 모드 (전기) 에서 필지 클릭 → ParcelInfoPanel 열리지만 폴리곤 음영은 안 그려질 가능성
- 공매 모드에서만 필지 폴리곤 빨간 음영 표시되는 것으로 보임
- 의뢰자는 **모드 무관** 필지 클릭 → 음영 + 패널 통일된 흐름 원함

### 구현 포인트
- `KakaoMap.tsx` 의 polygon 렌더링 로직을 모드 의존 → 모드 독립으로
- `selectedParcelPolygon` state 가 모든 모드에서 공유되도록
- 색상은 빨간 음영 통일 (의뢰자 명시) — `fillColor: rgba(239, 68, 68, 0.3)` 정도

## 2. 전기 + 공매 동시 표시 (현재 단일 모드 토글)

### 의뢰자 의도
- 지금은 "전기 모드" 또는 "공매 모드" 둘 중 하나만 활성
- 원하는 상태: **전기선로 마커 + 공매 매물 마커 동시 표시** (전기 + 공매 ON ON)
- 영업 시각: "이 동에 여유선로 충분 + 공매 매물 있다" 한 화면에서 즉시 판단

### 현재 상태 추정
- MapToolbar 의 [전기]/[공매] 버튼이 라디오 (배타적) 형태일 가능성
- 마커 컴포넌트도 활성 모드 1개에 따라 분기 렌더 가능성

### 구현 포인트
- 모드 = 단일 enum 상태 → **다중 활성 boolean 셋** (`{ kepco: true, onbid: true }`)
- KakaoMap 마커 layer 분리 (`kepcoMarkers`, `onbidMarkers`) — 둘 다 동시 렌더 가능
- 사이드바 패널은 어느 쪽? — 둘 다 토글 가능한 탭으로 통합 또는 마지막 활성 우선
- 색상 약속: 전기 = 파란색 계열 / 공매 = **빨간색** (의뢰자 명시)
- 마커 겹칠 때 클러스터링/오프셋 디자인 검토

## 3. 전기 + 경매 동시 표시 (경매 추가 후)

### 의뢰자 의도
- 경매 개발 완료 후, 전기 + 경매 동시 표시
- **경매 = 노란색 마커** (의뢰자 명시 — 공매와 색상 분리)

### 구현 포인트
- 경매 통합 (Hyphen API) 작업이 선결
- 경매 데이터 구조가 캠코 공매와 호환되도록 설계 → `source` 컬럼으로 구분 (공매 = 'onbid', 경매 = 'auction')
- 마커 layer 분리: `auctionMarkers` 추가
- 색상: 전기(파랑) / 공매(빨강) / 경매(노랑) — 3색 시각 구분

### 우선순위 의존성
- **선결**: 경매 통합 개발 (= Phase 2 항목 6)
- **그 다음**: 본 항목 3 (전기+경매 동시)
- 항목 1, 2 는 경매와 무관하게 즉시 착수 가능

## 우선순위 추천

1. **항목 1 (필지 클릭 → 음영)** — 즉시 작업 가능, 사용성 최대 개선
2. **항목 2 (전기 + 공매 동시)** — 즉시 작업 가능, 전체 UX 통일
3. **항목 3 (전기 + 경매 동시)** — 경매 개발 후 자연스럽게 추가

## 비용

- 항목 1, 2 = 시연 후 마무리 작업 (= 1차/2차 청구 범위 안에서 무상 보강)
- 항목 3 = 경매 개발(무상)에 포함
