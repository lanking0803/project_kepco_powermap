> 20년 원리금상환 10년 아니면 완전 자기자본 부담 / 견적서가 나와야 합니다.

→ **영업 매출 직결 도구**. SUNLAP 1차/2차/3차 (탐색 + 검증) 의 종착점 — "견적 들고 미팅".

## 🧮 비즈니스 로직 (전부 변수, 의뢰자 입력)

| 입력 | 기본값 | 비고 |
|---|---|---|
| **면적 → 패널 비율** | **2.5평당 1kW 1장** (고정) | 의뢰자 답: 항상 동일 |
| **kW 단가** | **130만원/kW** (1.3억/100kW) | 변수, 사용자 변경 가능 |
| **금리** | 변수 | 의뢰자 입력 |
| **상환 옵션** | 자기자본 / 20년 / 10년 (라디오 3) | 의뢰자 답: 3개로 충분 |
| **SMP 단가** (원/kWh) | 변수 (예시 PDF: 121.0) | 양식 PDF 분석 결과 |
| **REC 가중치** | 변수 (예시 PDF: 1.5) | 좌상단 시트명 "일출1.5" = REC 가중치 |
| **REC 단가** (원/REC) | 변수 (예시 PDF: 72.0) | |
| **일 발전시간** (h) | 변수 (예시 PDF: 4.0) | 지역별 다름 |
| **유지보수율** (연) | 3% (예시 PDF 고정) | |
| **첫해 효율 감소** | 0.99 | year 1→2 |
| **이후 연 효율 감소** | 0.996 | year 2~ |

### 자동 계산

```
패널 장수 = 면적(평) ÷ 2.5
총 kW    = 패널 장수 × 1kW (= 면적 ÷ 2.5)
연 발전량(kWh) = kW × 일발전시간 × 365 × 효율감소누적
SMP 수익 = 발전량 × SMP 단가
REC 수익 = 발전량 × REC 가중치 × REC 단가
총 수익 = SMP + REC
유지보수 = 총수익 × 3%
순수익 = 총수익 - 유지보수
대출 상환금 = PMT(금리, 상환기간, 시공비)
잔여수익 = 순수익 - 상환금
누적 잔여 = sum(잔여수익)
ROI = 누적 ÷ 초기투자 × 100
회수기간(년) = 초기투자 ÷ 평균순수익
```

## 📄 출력 양식 (의뢰자 제공, [docs/예시/](../../docs/예시/))

### 1. 견적서 PDF (A3 가로)
- 양식 파일: `태양광 예상견적서(봉남리6-2) A3.pdf`
- 구조: 좌측 = 20년 cash-flow 표 (년차/SMP/REC/총수익/유지보수/순수익/월평균/누적/대출상환/잔여)
- 우측 = 입력 변수 박스 (용량 kW, 일발전시간, SMP, REC, REC가중치) + 시공비 / 자기부담 / 대출액 / 금리(%) / 거치기간 / 상환기간 / ROI(%) / 회수기간(년) / 20년 누적순익

### 2. 배치도 PDF (별도)
- 양식 파일: `배치도(전라남도 구례군 구례읍 봉남리 6-2).pdf` 와 `배치도(전라남도 순천시 용당동 118-1).pdf`
- 구조:
  - 좌측 = 카카오맵 캡처 + 패널 그리드 시각화 (옥상/지상에 패널 N장 배치)
  - 우측 상단 = 회사 로고 (Solar LD inc) + 나침반 + 위치 정보 표
    - 부지 / 패널 모델 (예: AIKO 670Wp 2,465×1,134×33) / 패널 수량 / 총용량 / 이격거리 / 인버터
    - 회사정보 (전화 053-856-9698 / 050-4-370-2940 / slddettkh@naver.com)
    - Drawing By / Design By / Approved By / Project Title / Drawing Title

### 3. Excel (보류 — 견적서 관리 기능 추가개발 시)
